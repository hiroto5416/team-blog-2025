# Team Blog 2025

チーム開発のためのブログプラットフォーム

## 概要

このプロジェクトは、チーム開発における情報共有やコミュニケーションを促進するためのブログプラットフォームです。Next.jsとSupabaseを使用して構築されており、モダンなウェブ技術を活用した高速で安全なアプリケーションを提供します。

## 主な機能

- ユーザー認証・認可
- ブログ記事の作成・編集・削除
- マークダウンエディタ
- 記事のカテゴリ分類
- コメント機能
- 記事検索機能
- レスポンシブデザイン
